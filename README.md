# College-complaint-management-System
College complain management system is a software application that allows students, staff, and faculty to report and resolve any issue or grievances related to the college.
https://github.com/Dhanya-P-2003/College-complaint-management-System/tree/my-new-branch
College Complaint Management System is aimed at creating efficient software portal for addressing the grievances of the students for various purposes like broken furniture problem in class, microphone issues, projector problems, toilets plumbing, electricity issues, and other faculty mess related problems and many more.
